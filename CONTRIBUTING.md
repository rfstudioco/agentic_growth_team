# Contributing to the Agentic Growth Team

First off, thank you for considering contributing! It's people like you that make this such a great community.

We welcome contributions of all kinds, from fixing typos to adding brand new agents.

## How Can I Contribute?

### Reporting Bugs & Suggesting Enhancements
If you find a bug in an agent or have an idea for a new one, please open an issue. This is the best way to track the conversation. Please provide as much detail as possible.

### Pull Requests
Ready to contribute code or documentation?

1.  Fork the repository.
2.  Create a new branch for your feature or fix (`git checkout -b feature/your-new-feature`).
3.  Make your changes. Please ensure you update or add documentation as needed.
4.  Commit your changes with a clear message (`git commit -m 'feat: Add new agent for X'`).
5.  Push to your forked branch (`git push origin feature/your-new-feature`).
6.  Open a Pull Request against the `main` branch of this repository.

We'll review your PR, provide feedback, and merge it when it's ready. Thank you for your contribution!
